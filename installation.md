# Installation Details

Either method is fine! Both methods assume that Node.js and NPM are installed. Please use Node 22 or above. Please also make sure smartmontools (smartctl) is installed and the proper path is set in config.js if changed from default.

## Automatic Installation

Run the _install.bat at the top level.

## Manual Installation

Install the dependencies at the top level folder

```bash
npm i
```

Now you should be able to start the dev server and interact with the GUI.

```bash
npm run dev
```
